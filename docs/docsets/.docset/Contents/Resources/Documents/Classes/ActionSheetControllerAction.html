ActionSheetControllerAction

public class ActionSheetControllerAction

An ActionSheetControllerAction instance represents an action on the ActionSheetController that can be tapped by the user. It has a title or image for identifying the action and a handler which is called when the action has been tapped by the user.

  • image

    The action’s image. It will not be shown if the action also has a title.

    Swift

    public var image: UIImage?
  • init(style:title:image:dismissesActionController:handler:)

    Provide at least a title or an image. (But not both. The title overrides the image.) If you want the action to dismiss the controller when clicked set dismissesActionController to true.
